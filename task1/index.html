About This Project

This is Internship Task 1: API Integration. The dashboard fetches real-time weather data from the OpenWeatherMap API.

Features:

  • Current weather data
  • 5-day weather forecast
  • Search by city name
  • Geolocation support
  • Responsive design

API Information:

API: OpenWeatherMap

Endpoint: api.openweathermap.org/data/2.5/weather

Data: Temperature, humidity, wind speed, etc.
